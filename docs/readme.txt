Contains all documentation regarding the Quadratic Solver


Description
---------------
Quadsolver is a command line program for solving quadratic equations in the form of ax^2 + bx + c = 0, where a, b, and c are given
to the program by the user.


Insructions
---------------
To use program, simply enter in 3 IEEE Floating Point normalized values, each separated by a space, and then the program will take those numbers and output the resulting roots of those numbers.
